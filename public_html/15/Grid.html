Кто такой Bootstrap?

Bootstrap — это мощный, многофункциональный интерфейсный инструментарий.

Он содержит готовые шаблоны для кнопок, навигационных панелей, карточек, форм и множества других элементов, что позволяет быстро создавать современные и адаптивные сайты.

Где его найти и как скачать?

Есть несколько способов начать работу с Bootstrap:

  • Официальный сайт: Самый простой способ — зайти на официальную страницу getbootstrap.com и нажать кнопку "Download".
  • CDN-ссылки (самый популярный способ): Тебе даже не нужно скачивать файлы. Можно просто подключить готовые стили и скрипты в коде страницы.
  • Через пакетный менеджер: Если ты пользуешься npm или yarn, можно установить его командой npm install bootstrap.

Как это выглядит в коде?

Если ты выберешь способ с CDN, тебе нужно будет вставить в шапку сайта (внутри тега <head>) вот такую ссылку: 
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3/dist/css/bootstrap.min.css" rel="stylesheet" 
    integrity="sha384-sRIl4kxILFvY47J16cr9ZwB07vP4J8+LH7qKQnuqkuIAvNWLzeN8tE5YBujZqJLB" crossorigin="anonymous">
А перед закрывающимся тегом </body> вставить скрипты: 
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3/dist/js/bootstrap.bundle.min.js"
    integrity="sha384-FKyoEForCGlyvwx9Hj09JcYn3nv7wiPVlz7YYwJrWVcXK/BmnVDxM+D2scQbITxI" crossorigin="anonymous"></script>

Примечание: Версии могут меняться, лучше всегда брать актуальные ссылки на официальном сайте.

Система разметки Bootstrap

1. Контейнеры

В Bootstrap всё начинается с контейнера. Это базовый строительный блок, который содержит, выравнивает и центрирует содержимое страницы. Контейнеры бывают трех типов: фиксированный (.container), на всю ширину (.container-fluid) и адаптивные контейнеры (.container-sm, .container-md и т.д.).

<div class="container">
    <!-- Содержимое с отступами по бокам -->
</div>

<div class="container-fluid">
    <!-- Контейнер на всю ширину экрана -->
</div>

<div class="container-lg">
    <!-- Адаптивный контейнер (будет на всю ширину только на маленьких экранах) -->
</div>

2. Сетка

Bootstrap использует систему «12 колонок», 5 адаптивных ярусов, препроцессоров Sass и предустановленных классов.

Очень страшные слова, поэтому разберем на небольшом примере:

Представь, что ширина экрана разделена на 12 равных частей. 

<div class="row"> 
    <div class="col-4">Заняли 4 колонки</div>
    <div class="col-8">Заняли 8 колонок</div>
    <!-- Итого заняли 12 колонок -->
</div>

Почему именно 12? Число 12 удобно тем, что делится на много частей: на 2, 3, 4, 6. Это дает гибкость в верстке. Например, можно создать макет из двух колонок по 6, трех колонок по 4, четырех колонок по 3 или шести колонок по 2.

Адаптивные ярусы (брейкпоинты)

Адаптивные ярусы (брейкпоинты) — это точки перелома, при которых макет страницы меняется в зависимости от ширины экрана. Bootstrap имеет 6 предопределенных брейкпоинтов, основанных на размерах популярных устройств.

Классы .row-cols-*

Классы .row-cols-* — это специальные классы Bootstrap, которые задают количество колонок в ряду прямо на родительском контейнере .row. Вместо того чтобы писать класс для каждой колонки отдельно, мы пишем один класс на весь ряд. Это особенно удобно при работе с динамическим контентом, когда количество элементов может меняться.

Синтаксис:

  • .row-cols-{количество} — одинаковое количество колонок на всех экранах
  • .row-cols-{breakpoint}-{количество} — адаптивное количество колонок
  • .row-cols-auto — колонки подстраиваются под ширину содержимого

Sass (Syntactically Awesome Style Sheets)

Sass — это препроцессор CSS, который расширяет возможности обычного CSS. Он позволяет использовать переменные, вложенность, миксины и другие возможности, делая написание стилей более удобным и поддерживаемым. Bootstrap написан на Sass, поэтому его легко настраивать под свои нужды.

Основные возможности Sass:

  • Переменные — можно задать значения (цвета, отступы) один раз и использовать их многократно
  • Вложенность — позволяет писать стили в иерархическом виде, как в HTML
  • Миксины — переиспользуемые блоки CSS-кода
  • Наследование — один селектор может наследовать стили другого
Обычный CSS
.button {
    background-color: blue;
    color: white;
    padding: 10px;
}

.button-large {
    background-color: blue;
    color: white;
    padding: 20px;
}
Sass
$main-color: blue;
$text-color: white;

.button {
    background-color: $main-color;
    color: $text-color;
    padding: 10px;
    
    &-large {
        padding: 20px;
    }
}

В примере выше видно, как Sass позволяет избежать повторения кода. Мы создали переменные для цветов и использовали вложенность (&-large) для создания модификатора кнопки. Это делает код более чистым и легким в поддержке.

Переменные сетки в Sass

Bootstrap использует следующие переменные для настройки сетки:

// Количество колонок
$grid-columns: 12;

// Отступы между колонками (gutter)
$grid-gutter-width: 1.5rem;

// Максимальное количество для .row-cols-*
$grid-row-columns: 6;

// Контрольные точки (breakpoints)
$grid-breakpoints: (
  xs: 0,
  sm: 576px,
  md: 768px,
  lg: 992px,
  xl: 1200px,
  xxl: 1400px
);

// Ширина контейнеров
$container-max-widths: (
  sm: 540px,
  md: 720px,
  lg: 960px,
  xl: 1140px,
  xxl: 1320px
);
Миксины для создания своей сетки
// Создание контейнера
.my-custom-container {
    @include make-container();
}

// Создание ряда
.my-custom-row {
    @include make-row();
}

// Подготовка колонки
.my-custom-col {
    @include make-col-ready();
    @include make-col(6); // колонка на 6 частей
}

// Колонка со смещением
.my-col-offset-3 {
    @include make-col-offset(3);
}

Настройка сетки под себя

Вы можете изменить параметры сетки, переопределив переменные Sass до подключения Bootstrap:

// Например, сделать 10-колоночную сетку
$grid-columns: 10;

// Уменьшить отступы между колонками
$grid-gutter-width: 1rem;

// Добавить свою контрольную точку
$grid-breakpoints: (
  xs: 0,
  sm: 576px,
  md: 768px,
  lg: 992px,
  xl: 1200px,
  xxl: 1400px,
  xxxl: 1600px  // новая точка
);

// Затем подключить Bootstrap
@import "bootstrap";

Примечание: После изменения переменных нужно перекомпилировать Sass в CSS.

Вложенность сетки

Вложенность — это возможность создавать новую сетку внутри существующей колонки. Это позволяет строить сложные макеты с дополнительной структурой внутри основных блоков. Например, внутри основной колонки с контентом можно создать еще один ряд с несколькими колонками.

Важное правило: Вложенные колонки считаются относительно родительской колонки, а не всей страницы. Если родитель занимает 8 колонок из 12, то внутри него колонка с классом col-6 будет занимать половину от этих 8 колонок (то есть 4 колонки от общей ширины страницы). Сумма вложенных колонок не должна превышать 12, но может быть меньше.

Основная колонка (8/12)

Левая подколонка (6/12 внутри родителя)
Правая подколонка (6/12 внутри родителя)

Внутри создан новый ряд с двумя колонками по 6

Боковая колонка (4/12)

В данном примере основная колонка занимает 8 частей из 12. Внутри нее создан новый ряд с двумя колонками, каждая из которых занимает по 6 частей внутри родителя. Таким образом, левая и правая подколонки фактически занимают по 4 части от общей ширины страницы (половина от 8).
