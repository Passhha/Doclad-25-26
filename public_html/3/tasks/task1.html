Создайте форму обратной связи

Форма должна содержать:

  • обязательное поле для ввода имени
  • поле для ввода email
  • поле для текста сообщения
  • кнопку отправки формы
  • также необходимо использовать label для подписей к элементам формы

Подсказка:

  • Для визуального обозначения поля как обязательного, можно использовать: < div class="requiredForm">
